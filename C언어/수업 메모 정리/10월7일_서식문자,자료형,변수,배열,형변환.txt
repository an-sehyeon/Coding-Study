 서식문자

 서식이란?		: 서류를 꾸민는 일정한 양식
 서식문자란? 	: 서식공간에 자료를 대입하여 화면에 출력하거나 입력받기 위한 용도의 문자

 %d : decimal의 약자 	10진수(정수) 서식문자
 %f : flot의 약자 		10진수(실수) 서식문자
 %c : character의 약자 	한문자 서식문자
 %s : String의 약자	문자열 서식문자
 %o : Octal의 약자 	8진수(정수) 서식문자
 %x : Hexadecimal의 약자	16진수(정수) 서식문자
 %p : 포인터의 주소값 출력

 #include <stdio.h>
 int main() {
	printf("정수 서식문자 : %d\n", 100);				// 서식문자 %d를 통해 정수인 100을 출력
	printf("정수 덧셈 : %d + %d = %d\n", 200, 10, 200 + 10);		// 위와 같이 %d의 서식문자를 사용하여 정수 200,10,(200+10)의 값인 210의 값을 출력	
	printf("실수 서식문자 : %f\n", 100001.1111111111);		// %f는 실수형 서식문자로 소수점 값을 출력	
	printf("한문자 서식문자 : %c\n", 'B');				// %c는 한문자 서식문자로 한 단어 출력
	printf("문자열 서식문자 : %s\n", "Good Morning");		// %s는 문자열을 출력	
	printf("\n");

	return 0;

  }

	printf("8진수 : %x\n", 0255);
	printf("8진수 : %x\n", 0xad);
	printf("\n");

	return 0;
}

 [ 자료형과 변수 ]

 1byte = 8 bit

 자료(데이터) : 선 저장 후 처리 ( 공간 먼저 확보 후 저장 )
 자료형 : 데이터 종류에 따라 저장될 공간을 미리 규격화 시켜놓은 예약어

 자료형의 종류

		int	( 4 byte )
		short	( 2 byte )
		char	( 1 byte )



 2. 실수 자료형 : long double ( 8 byte 혹은 그 이상 )
		( 소수점 이하 15자리 또는 그 이상 표현 가능 )
		
		double ( 8byte )
		(소수점 이하 15자리까지 표현 가능 )
		
		float ( 4byte )
		(소수점 이하 8자리까지 표현 가능 )



 3. 문자 자료형 : char ( 1byte )


 4. 문자열 : 배열 혹은 포인터를 사용하여 처리

문제 01

korean 	변수를 90으로 초기화
math 	변수를 100으로 초기화
english 	변수를 35으로 초기화



mom_age	 변수를 50으로 초기화
my_age	 변수를 22으로 초기화




total 	변수를 선언하고 korean, math, english의 합을 대입
min 	변수를 선언하고 mom_age와 my_age의 차를 대입



printf()함수와 서식문자(%d)를 이용해서 total과 min를 출력


[결과값]
국영수 성적의 총합은 90 + 100 + 35 = 255입니다.
엄마와 나의 나이 차는 50 -22 = 28살입니다.


#include <stdio.h>	
int main() {
	int korean = 90;			// int형 korean이라는 변수명에 90이라는 값 대입
	int math = 100;			// int형 math이라는 변수명에 100이라는 값 대입
	int english = 35;			// int형 english이라는 변수명에 34이라는 값 대입


	int mom_age = 50;			// int자료형, mom_age라는 변수명에 50의 값 대입
	int my_age = 22;			// int자료형, my_age라는 변수명에 22의 값 대입


	int total = korean + math + english;	// int자료형, korean, math, english 세개 변수의 담긴 값을 더한 값을 total이라는 변수에 대입	
	int min = mom_age - my_age;	// int자료형, mom_age의 값과 my_age의 값의 뺀 값을 min이라는 변수에 대입	


	printf("국영수 성적의 총 합은 %d + %d + %d = %d입니다.\n", korean, math, english, total);
	printf("엄마와 나의 나이 차는 %d - %d = %d입니다.", mom_age, my_age, min);
	printf("\n");
	
	return 0;

}

 [자료형변환]
 
 자료형 변환의 종류

 1. 자동형변환 (암시적 형 변환) : 컴파일러가 자동으로 형 변환해주는 경우
 2. 강제형변환 (명시적 형 변환) : 개발자(프로그래머)가 인위적으로 강제로 형 변환하는 경우



------------------------------------------------------------------------------------------------------------------ -

 [자동형변환]

 범위가 넓은 자료형에 범위가 좁은 자료형이 대입되는 경우
 컴파일러가 알아서 형 변환을 해준다는건 데이터 손실 가능성이 낮거나 없을 때라는 것임.

 double는	 	 float 보다	범위가 넓음.
 float는		 long long 보다	범위가 넓음.
 long long는	 int 보다	범위가 넓음.
 int는		 short 보다	범위가 넓음.
 short는		 char 보다	범위가 넓음.

 즉, 이는 데이터 손실이 최소화된다는 의미.
 EX) 큰공간(double) = 작은값(10)

 [강제형변환]

 int 약 -21억 ~ +21억까지
 short : 약 -32000 ~ +32000.
 char : -128 ~ +127

 프로그래머(개발자)가 특정 자료형을 강제로 형 변환하는 경우
 데이터 손시르이 가능성 0
 당연히 좁은 범위의 자료형에다가 큰 값을 넣으려는 경우
