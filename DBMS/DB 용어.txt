select 출력할테이블.칼럼명, 조인할테이블.칼럼명
왼쪽테이블 left outer join 오른쪽 테이블
on 조인할왼쪽테이블.칼럼명 = 조인할오른쪽테이블.칼럼명
				=왼쪽조인(필요한 칼럼만 조인)

select * from 테이블명 left join 테이블명 on 
테이블명.조건이같은칼럼명 = 테이블명.조건이같은칼럼명;
				=2개의 테이블 통으로 조인	
---------------------------------------------------------------------
select 출력할테이블.칼럼명, 조인할테이블.칼럼명
오른쪽테이블 right outer join 왼쪽 테이블
on 조인할오른쪽테이블.칼럼명 = 조인할왼쪽테이블.칼럼명
					=오른쪽조인
---------------------------------------------------------------------
select 출력할 테이블.컬럼명, 출력할테이블.컬럼명
from 조인할 테이블, 조인할테이블
where 조건조회할 테이블.컬럼명 = 조건조회할 테이블.컬럼명;
					=이너조인
---------------------------------------------------------------------
select 출력할 테이블.컬럼명, 출력할테이블.컬럼명
from 조인할 테이블, 조인할테이블
where 조건조회할 테이블.컬럼명 = 조건조회할 테이블.컬럼명
order by 정렬 기준 테이블 오름또는내림;
					=조인 후 정렬
