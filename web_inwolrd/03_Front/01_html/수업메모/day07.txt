1. Advanced <a>

    visited : 한 번이라도 가본 적 있음
    hover : 마우스를 올려 놓으면
    active : 마우스를 클릭하고 있으면

    페이지 책갈피
        현재 페이지에서 이동하고 싶은 곳으로 이동하는 법
        <a> 태그의 id 속성을 이용하여 책갈피 생성


2. iframe
    - 외부 사이트를 가져와서 링크
    - 내 페이지 안에 있는 것처럼 표현

    - 이 기능을 활용하면 다수의 웹 페이지(남이 만든)를 내가 만든 것처럼 한 웹 페이지 안에 다 넣을 수 있음

    - 일부 웹 페이지는 도용을 막기 위해 iframe 지원 금지


3. Semantic Tag
    - 기존 방식은 body 내에 수 많은 div로 이루어져 있었음
    - div : division의 약자로 별다른 의미는 없는 구역이라는 의미

    - 각 구역을 각각의 용도에 맞도록 새롭게 네이밍
    - 대표적인 구역
        header : 페이지 상단부 공통
        navigation : 내비게이션 메뉴
        aside : 타 사이트 링크 등의 외부 링크
        footer : 페이지 하단부 공통 ex) 회사소개, copyright

        section : 문서를 주제별로 분류
        article : 독립된 기사, 내용이 독립적이고 이 부분만 떼어서 옮겨도 자연스러움

        => section과 article의 애매모호함으로 div가 강화되고 id 등으로 명확하게 의미를 표시
