사이트명 	
 - MindGoal(마인드골) 


사이트의 개요 - 무엇을 하는 사이트 인지?
 - 주제 : 스포츠 심리 상담
 - 목적 : 
	- 운동선수, 학부모, 지도자 등이 자신의 심리 상태를 진단하고, 결과를 기반으로 맞춤형 상담을 받을 수 있는 플랫폼 제공.
	- 개인적인 고민이나 현재 상황에 대한 전문적인 심리 상담 지원.
	
	- 주요 기능 : 
		    1. 심리 진단 테스트 : 사용자가 현재 심리 상태를 파악할 수 있도록 과학적이고 체계적인 테스트 제공
		    2. 결과 분석 및 방향 제시 : 테스트 결과를 바탕으로 사용자의 심리적 상태를 분석하고, 이를 기반으로 개선 방향과 조언 제공.
		    3. 상담 신청 : 사용자가 상담 신청을 통해 전문가에게 전달하고 상담을 예약
		    4. 전문가 전용 페이지 : 상담사가 고객의 테스트 결과와 상담 신청 내용을 확인하고 관리할 수 있는 전용 페이지 제공
	
	- 대상 사용자 : 
		    1. 운동선수 : 경기력 향상과 심리적 안정이 필요한 선수.
		    2. 학부모 : 자녀의 심리적 상태를 이해하고 올바르게 지원하고자 하는 부모.
		    3. 지도자 : 팀 관리와 선수들과의 소통을 원활히 하기 위한 심리적 도구가 필요한 지도자.


사이트의 특징 - 사이트가 갖는 장점 (기존 유사 사이트에 비해)
1. 심리 진단과 상담의 연계성 강화
 - 기존 사이트는 심리 진단과 상담이 분리되어 있는 경우가 많지만 사용자가 진단 결과를 바탕으로 바로 상담 신청으로 이어지는 
   원스톱 솔류션 제공

2. 대상별 맞춤형 서비스 제공
 - 운동선수, 학부모, 지도자 등 사용자 그룹별로 최적화된 콘텐츠와 테스트 제공.
 - 다양한 사용자 맞춤형 접근 방식
   ex) 선수는 경기력과 멘탈 강화에 초점, 학부모는 자녀 지원 방법에 초점.
