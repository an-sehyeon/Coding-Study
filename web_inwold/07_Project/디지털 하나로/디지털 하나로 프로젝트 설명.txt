스포츠 심리 상담 플랫폼을 개발하는 웹사이트를 기획부터 개발까지 전적으로 혼자서 진행하였습니다. 이 웹사이트는 사용자가 심리 진단 테스트를 수행하고, 온라인 상담을 신청할 수 있도록 설계되었습니다. 현재까지 회원가입, 로그인, 보안 관리, 관리자 페이지, 기본적인 게시판 기능을 구현하였으며, 심리 진단 테스트 및 상담 신청 기능을 추가하며 지속적으로 프로젝트를 확장해 나가고 있습니다.

[ 주요 기능 및 구현 내용 ]
1. 사용자 인증 및 보안
- Spring Security 기반의 인증 및 접근 제어
- /main, /login, /join, /board 페이지는 인증 없이 접근 가능
- /admin 페이지는 관리자(Admin) 권한을 가진 사용자만 접근 가능
- 동일 계정 다중 로그인 차단 및 세션 고정 보호 기능 적용
- BCryptPasswordEncoder를 사용하여 비밀번호를 안전하게 암호화 후 저장

2. 회원가입 기능
- 회원가입 폼을 통해 사용자 정보를 DB에 저장
- 아이디 및 이메일 중복 검사 및 비밀번호 유효성 검사 적용
- 사용자 유형 선택 (선수, 학부모, 지도자) 및 검증
- admin 역할을 사용자가 직접 선택할 수 없도록 제한

3. 로그인 및 세션 관리
- 사용자 ID와 비밀번호를 검증하여 로그인 처리
- 사용자 역할(일반 사용자/관리자)에 따라 메인 페이지 UI 차별화
- 비로그인 상태의 사용자는 anonymousUser로 처리하여 특정 기능 제한
- 로그아웃 시 세션 무효화 및 쿠키 삭제 후 메인 페이지로 리디렉션

4. 관리자 페이지
- 관리자 계정만 접근 가능하도록 권한 설정

5. 게시판 기능
- 게시글 CRUD 기능 구현 (작성, 조회, 수정, 삭제 가능)
- 로그인한 사용자만 글 작성 가능하도록 접근 제어 적용

6. 메인 페이지
로그인 후, 회원가입 시 입력한 사용자의 이름과 환영 문구 표시

[ 향후 구현할 기능 ]
1) 심리 진단 테스트 기능
2) 사용자의 심리적 상태를 분석하여 결과를 제공
3) 온라인 상담 신청 및 예약 기능
5) 관리자 페이지 확장(회원 및 상담 내역 관리 기능 추가)
7) 마이페이지 기능 구현(사용자가 로그인 후, 상담 신청 내역 및 작성한 게시글 확인 가능)
9) 게시판 기능 강화
10) 관리자만 특정 게시글 삭제 가능하도록 제한
11) 게시글을 작성한 사용자와 관리자만 해당 글을 확인할 수 있도록 설정
