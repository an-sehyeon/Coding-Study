프로젝트 순서

1. 팀 구성
2. 주제 선정

3. 화면 역할 
    (1) 로그인(회원가입)
    (2) 관리자(로그인)
    (3) 심리 진단 테스트 화면
    (4) 관리자 전용 화면
    (5) 온라인 상담 화면(게시판)
    (6) 게시판 글 클릭 시 게시글 화면
    (7) 관리자 창에서 결과+문의글 클릭 시 화면
    (8) "홈" 버튼 클릭 시 메인 화면으로 이동
    

4. 화면별 기능
        
    (1) 로그인 화면             - 로그인 기능, 로그인 성공 시 메인 화면으로 이동 + "회원명"님 반갑습니다 알림창 출력
                               - 회원가입 
                               - 아이디 찾기
                               - 비밀번호 재설정

    (2) 회원가입 화면           - 회원정보입력   
                                    - 이름
                                    - 생년월일 
                                    - 아이디    "중복확인"
                                    - 비밀번호
                                    - 비밀번호 재입력
                                    - 연락처
                                    - 이메일
                                    - 주소
                                    - 소속
                                    - 대상 선택 (선수 + 학부모 + 지도자)     
                                    - "회원가입" 클릭 시 회원 정보 데이터 저장 

    (3) 회원가입 완료 화면      - 로그인 페이지로 이동 + "회원가입이 완료되었습니다" 문구 출력                                                          
                   
    (3) 관리자(로그인)          - 로그인 기능

    (4) 심리 진단 테스트 화면   - 대상에 따른 심리 질문지 업로드
                               - 고객이 질문지 작성 후 "제출" 클릭 시 결과 도출(그래프+피드백 출력)
                               - 상담사에게 문의 내용 작성 후 "상담신청" 클릭 시 진단결과 + 문의 내용 상담사에게 전송

    (5) 관리자 전용 화면        - (3)번에서 전달된 내용 리스트 업로드

    (6) 온라인 상담 화면(게시판) - 게시글 
                                - "작성하기"
                                - 검색 기능
                                
    (7) 게시판 글 클릭시 게시글 화면 - 제목 + 내용
                                   - "수정" + "삭제" "목록" + "저장"
                                   - "수정" + "삭제" 클릭 시 비밀번호 입력(작성자만 가능)

    (8) 관리자 창에서 결과+문의글 클릭 시 화면 - 질문지 내용 + 결과 + 문의 사항 + 회원 정보
