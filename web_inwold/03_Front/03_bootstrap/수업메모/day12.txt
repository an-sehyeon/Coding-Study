1. 부트스트랩(BootStrap)
    - 세계에서 가장 인기있는 모바일 우선형/반응형 프론트엔드 프레임워크
    - 프레임워크
        패키지 : 소스 + 디렉토리
        라이브러리 : 여러 개의 패키지
        프레임워크가 제공하는 공간에 라이브러리 탑재 가능
    
    - 가장 빈번하게 사용되는 CSS, JS를 미리 만들어놓고 CDN(Content Delivery Network)이나 다운로드 방식으로 사용하도록 서비스

    (1) 설치 방법
        - 설치를 따로 하지는 않으며, getbootstrap.com 사이트에서 다운로드 > 압축 해제
        - CSS의 경우 link 방식으로 사용


    (2) 탄생 배경
        - 트위터 프로젝트에서 시작
        - 서로 다른 인터페이스를 사용한 여러 개발자의 공동 작업
        - 디자인 불일치, 관리 난이도 상승, 코드 방대 -> 일관성 관리 불가능
        - 트위터 개발자와 UI 디자이너가 최초로 개발


    (3) 부트스트랩
    - Frontend 개발을 빠르고 쉽게 할 수 있는 프레임워크
    - HTML, CSS 기반의 템플릿 양식, 버튼, 내비게이션 및 기타
    - 트위터에서 사용하는 각종 레이아웃, 버튼, 입력창 등 디자인을 CSS, JS로 구성
    - 프레임워크이므로 jQuery도 사용 가능
    - 상업적 이용 가능
    - 모바일/PC/태블릿 디자인 지원
    - 검색 엔진을 통해 (free) bootstrap template으로 검색하면 엄청난 양의 데이터가 검색됨



2. Container
    - 내용물을 담는 그릇
    - class="Container" : 모든 breakpoint마다 설정 변경
    - class="container-{bp}" : bp보다 작아지면 100%
        ex) container-md : md(768px)보다 작아지면 100%
                           md(768px)보다 크면 bp마다 설정 변경
    - class="container-fluid" : 그냥 100%



3. Grid
    - 테이블과 유사한 형태로 화면을 분할, 레이아웃 설정
    - 기본은 12컬럼 : 가장 약수가 많은 숫자
                2, 3, 4, 6, 8, 9
    - 5개의 Grid 옵션
        .col-xs-{}
        .col-sm-
        .col-md-
        .col-lg-
        .col-xl-
