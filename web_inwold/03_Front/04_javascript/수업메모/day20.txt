2. DOM(Document Object Model)
    - HTML Tag 들을 하나씩 오브젝트화 한 것
    - HTML 페이지의 모양과 내용을 제어하기 위해 사용하는 객체

    <div>
        <ul>
            <li></li>
        </ul>
    </div>

    - div, ul, li 어떤 것도 자바스크립트의 객체로 선언 가능
    - HTML 태그 당 DOM 객체가 하나씩 생성
    - HTML 태그의 포함관계에 따라 Parent-Child 관계로 구성
